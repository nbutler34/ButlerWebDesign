Program

  1. Open Adobe Illustrator
  2. Create a new document in landscape orientation that is 1920 by 1080 pixels
  3. Save the project to your desktop
  4. Fill the background with a rectangle
  5. Change the rectangle's fill color to a hex value of #2A2B33
  6. Create a circle in the center of the page with a diameter equal to half the height of the page
  7. Change the circle's fill color to a hex value of #E89010
  8. Create a second circle in the center of the space to the right of the first circle with a diameter equal 1/4 of the first circle's diameter
  9. Change the second circle's fill color to a hex value of #4C3517
  10. Save the file
  11. Choose File > Export > Export for Screens
  12. Set "Export to" as your desktop, make sure the scale is set to 1x and the format is set to PNG before clicking "Export Artboard"
  13. Send the exported image to nbutler6@masonlive.gmu.edu
